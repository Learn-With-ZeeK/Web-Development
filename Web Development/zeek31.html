How selectors Work

div p:
It works when there is a para anywhere inside the division.

div li p:
It works when there is a list element anywhere inside div and inside this list element there is a para anywhere.

div>p:
It works when there is a para directly under div.

div+p:
It works when there is para after div block ends.
