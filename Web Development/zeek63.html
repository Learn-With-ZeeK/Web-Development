How to use node js in VSC?

  1. Open visual studio code by going inside Complete Web Development Part .
  2. Node.js has been installed and working already.
  3. Write your code you want to run in html file.
  4. Create a js file.
  5. In js file click on terminal(on top between run and help)
  6. Now open new terminal.
  7. This will appear PS C:\Users\abc\Desktop\webpart\Complete Web Development Part>
  8. Write node zeek63.js
  9. Code will work. see that by using console.log("abc");.
  10. Go to node js download website and go to about.Link is given. CLICK HERE
  11. Copy the code.....
    const http = require('http'); const hostname = '127.0.0.1';
    const port = 3000;
    const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello World');
    });
    server.listen(port, hostname, () => {
    console.log(`Server running at http://${hostname}:${port}/`);
    });
  12. In res.end(``) Copy your entire html file in between ``.
  13. In[res.setHeader('Content-Type', 'text/plain'); ] change plain to html.
  14. Now your page runs on 127.0.0.1
  15. Remember to delete or clear your terminal because one hostname one page only.
