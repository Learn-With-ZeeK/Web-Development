Events and Event listening in JavaScript

The change in the state of an object is known as an Event. When javascript code is included in HTML, js react over these events and allow the execution. This process of reacting over the events is called Event Handling.Thus, js handles the HTML events via Event Handlers.
    Mouse Events
    Event Performed..../Event Handler..../Description
  • click ..../onclick..../When mouse click on an element
  • mouseover..../onmouseover..../When the cursor of the mouse comes over the element
  • mouseout..../onmouseout..../When the cursor of the mouse leaves an element
  • mousedown..../onmousedown..../When the mouse button is pressed over the element
  • mouseup..../onmouseup..../When the mouse button is released over the element
  • mousemove..../onmousemove..../ When the mouse movement takes place.
Event Handler is onEvent-Performed.For Example- click is Event-Performed and onclick is Event Handler.
Some other events like mouse events are form events(focus/submit/blur/change),keyboard events (Keydown & Keyup)and Window Events(load/unload/resize).
