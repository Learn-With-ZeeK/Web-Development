Use of alert(),prompt() and confirm()

What is the use of alert in JavaScript? The alert() method in JavaScript displays an alert dialog box that consists of some specified message (which is optional) and an OK button. When the dialog box pops up, we have to click "OK" to proceed.

What does function prompt () do? The prompt() method is used to display a dialog with an optional message prompting the user to input some text

What is confirm method? The confirm() method is used to display a modal dialog with an optional message and two buttons, OK and Cancel. It returns true if the user clicks “OK”, and false otherwise.
